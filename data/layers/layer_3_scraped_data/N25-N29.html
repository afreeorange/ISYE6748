Other disorders of kidney and ureter N25-N29

Type 2 Excludes
Type 2 Excludes Help
A type 2 excludes note represents "not included here". A type 2 excludes note indicates that the condition excluded is not part of the condition it is excluded from but a patient may have both conditions at the same time. When a type 2 excludes note appears under a code it is acceptable to use both the code (N25-N29) and the excluded code together.
