Hemorrhagic and hematological disorders of newborn P50-P61

Type 1 Excludes
Type 1 Excludes Help
A type 1 excludes note is a pure excludes. It means "not coded here". A type 1 excludes note indicates that the code excluded should never be used at the same time as P50-P61. A type 1 excludes note is for used for when two conditions cannot occur together, such as a congenital form versus an acquired form of the same condition.
