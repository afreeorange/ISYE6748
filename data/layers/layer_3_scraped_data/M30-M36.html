Systemic connective tissue disorders M30-M36

Type 1 Excludes
Type 1 Excludes Help
A type 1 excludes note is a pure excludes. It means "not coded here". A type 1 excludes note indicates that the code excluded should never be used at the same time as M30-M36. A type 1 excludes note is for used for when two conditions cannot occur together, such as a congenital form versus an acquired form of the same condition.
  • autoimmune disease, single organ or single cell-type -code to relevant condition category
Includes
Includes Help
"Includes" further defines, or give examples of, the content of the code or category.
  • autoimmune disease NOS
  • collagen (vascular) disease NOS
  • systemic autoimmune disease
  • systemic collagen (vascular) disease
