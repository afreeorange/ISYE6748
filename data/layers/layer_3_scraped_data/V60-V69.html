Occupant of heavy transport vehicle injured in transport accident V60-V69

Type 1 Excludes
Type 1 Excludes Help
A type 1 excludes note is a pure excludes. It means "not coded here". A type 1 excludes note indicates that the code excluded should never be used at the same time as V60-V69. A type 1 excludes note is for used for when two conditions cannot occur together, such as a congenital form versus an acquired form of the same condition.
  • bus
  • motorcoach
Includes
Includes Help
"Includes" further defines, or give examples of, the content of the code or category.
  • 18 wheeler
  • armored car
  • panel truck
