Dermatitis and eczema L20-L30

Note
  • In this block the terms dermatitis and eczema are used synonymously and interchangeably.
Type 2 Excludes
Type 2 Excludes Help
A type 2 excludes note represents "not included here". A type 2 excludes note indicates that the condition excluded is not part of the condition it is excluded from but a patient may have both conditions at the same time. When a type 2 excludes note appears under a code it is acceptable to use both the code (L20-L30) and the excluded code together.
